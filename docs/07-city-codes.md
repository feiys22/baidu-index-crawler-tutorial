# 全国城市代码对照表

> 百度指数的城市代码，用于 API 请求中的 `area` 参数。

---

## 一、什么是城市代码？

百度指数 API 中的 `area` 参数接受一个**数字代码**，对应不同的城市或省份。

例如：
- `area=0` → 全国（所有城市合计）
- `area=911` → 北京
- `area=910` → 上海

**注意：这里的代码是百度自己的内部编码，不是行政区划代码（那种 6 位数字的）。**

---

## 二、全国 144 个城市代码表

使用方法：在代码里找到对应城市，把代码填入即可。

```python
# 示例
area = 911  # 北京
area = 910  # 上海
area = 94   # 深圳
```
